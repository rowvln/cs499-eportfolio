# Professional Self-Assessment

Throughout the Computer Science program at SNHU, I’ve developed both the technical and professional skills needed to pursue a career in software development. The courses I completed helped me build a strong foundation in programming, software engineering, databases, algorithms, security, and full-stack development. Creating this ePortfolio gave me the opportunity to reflect on how much I’ve grown throughout the program and how the skills I developed connect to solve real-world problems.

One of the biggest things I learned throughout the program is that successful software development is about more than just writing code. Collaboration and communication play a major role in creating solutions that meet user and business needs. Through both my coursework and my experience working as a Product Manager with Develop for Good, I learned how important it is to work with different stakeholders, gather requirements, listen to feedback, and communicate technical ideas to people with different backgrounds. These experiences helped me better understand how software projects are developed in professional environments and how teamwork contributes to better decision-making.

My communication skills also improved throughout the program. Many assignments required written reports, presentations, code reviews, and technical documentation. Learning how to explain technical concepts clearly became just as important as implementing the solutions themselves. Developing this ePortfolio allowed me to organize my work in a professional way while demonstrating both my technical abilities and my ability to communicate those skills effectively.

Throughout the program, I gained experience with software engineering principles and full-stack development practices. Courses such as CS 250, CS 320, and CS 465 helped me understand software architecture, testing, maintainability, and the software development lifecycle. These skills are demonstrated throughout the Travlr Getaways project that serves as the foundation of this portfolio. By enhancing the project across multiple categories, I was able to improve validation, authentication, maintainability, and overall application design while gaining a deeper understanding of professional software development practices.

I also strengthened my understanding of algorithms and data structures. Before entering the program, I primarily focused on making applications function correctly. As I progressed through my coursework, I learned the importance of efficiency, scalability, and selecting the right approach for processing data. In the Travlr Getaways enhancements, I implemented search functionality, sorting, pagination, and database indexing to improve performance and reduce unnecessary processing. These improvements helped me better understand how algorithmic thinking can improve real-world applications.

Database design and management became another important area of growth. Through courses such as DAD 220 and CS 340, I learned how databases support software applications and how proper data management contributes to reliability and performance. During the database enhancement for Travlr Getaways, I implemented additional validation rules, indexing strategies, and data integrity controls that improved both the reliability and efficiency of the application. This experience helped reinforce the importance of designing databases that are secure, scalable, and maintainable.

Security is another area that became increasingly important throughout the program. I learned that security shouldn’t be treated as a separate feature but instead should be considered throughout the development process. Across my enhancements, I implemented stronger validation, authentication protections, controlled database queries, and error handling practices that help reduce vulnerabilities and improve application reliability. These improvements helped me develop a stronger security mindset and a greater appreciation for defensive programming practices.

The artifacts included in this ePortfolio demonstrate how the knowledge and skills I developed throughout the program work together to create a complete software solution. The code review highlights my ability to evaluate existing software and identify opportunities for improvement. Enhancement One focuses on software design and engineering through improvements to validation, authentication, maintainability, and application structure. Enhancement Two demonstrates algorithms and data structures through search functionality, sorting, pagination, and performance improvements. Enhancement Three focuses on database development through schema validation, indexing, query optimization, and data integrity controls. Together, these enhancements show how different areas of computer science contribute to building reliable and scalable applications.

Overall, completing the Computer Science program and developing this ePortfolio has strengthened both my technical knowledge and my confidence as a developer. The combination of coursework, hands-on projects, and real-world collaboration experiences has prepared me to continue growing as a software professional. This portfolio represents not only the projects I completed throughout the program but also the skills, lessons, and experiences that have shaped my development as a computer scientist.
