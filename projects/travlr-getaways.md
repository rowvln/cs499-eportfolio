# Travlr Getaways

## Project Overview

Travlr Getaways is a full-stack travel management application originally developed during CS 465 Full Stack Development.

Technologies used:

- Angular
- Node.js
- Express
- MongoDB
- Mongoose

The application includes:

- Customer-facing travel website
- Administrative portal
- Authentication
- CRUD functionality
- RESTful API integration

For CS 499, I enhanced this application in three categories.
