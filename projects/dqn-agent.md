# Deep Q-Learning Agent

## Description

This project was originally developed in CS 370 and focuses on reinforcement learning through the implementation of a Deep Q-Network (DQN).

I selected this project because it represents one of the most technically challenging projects completed throughout the Computer Science program and provides a strong opportunity to demonstrate algorithmic thinking, performance evaluation, and problem solving.
